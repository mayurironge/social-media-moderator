# AI Social Media Moderation Platform

## Overview

An AI-powered social media moderation platform that detects harmful content using a rule-based engine, machine learning models, and generative AI.

## Features

- Rule-based moderation
- Toxicity detection
- Hate speech detection
- Threat detection
- Spam detection
- PostgreSQL database
- FastAPI backend
- React frontend
- Analytics dashboard
- Future Reddit/X/Facebook/Instagram connectors

## Tech Stack

### Backend
- Python
- FastAPI
- SQLAlchemy
- PostgreSQL

### Frontend
- React
- Vite

### AI
- Rule-based NLP
- Hugging Face Transformers (planned)
- Generative AI (planned)
